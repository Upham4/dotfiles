#Homefiles

##Files

### .bash.aliases
- gs = git status
- ga = git add
- gc = git commit "m"
- gco = git checkout
- gpl = git pull origin

### install_bash.sh
- Script that copies the bash.aliases file to the home directory then tells the user to use source ~/.bash_aliases to activate the aliases 

## Sources
- [Awesome bash aliases](https://github.com/ctrlaltvikas/awesome-bash-alias) where i got the bash aliases from
- [Stack Overflow](https://stackoverflow.com/questions/11548093/my-bash-aliases-dont-work) helped with the script 
- [GNU Bash Manual — Aliases](https://www.gnu.org/software/bash/manual/html_node/Aliases.html) helped with aliases
